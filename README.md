# yahoo-finance-cli

NodeJS CLI to utilize the Yahoo Finance CSV API.

See [Yahoo! APIs Terms of Use](https://developer.yahoo.com/terms/)

## Install

`npm install -g https://github.com/tabrindle/yahoo-finance-cli`
OR
`yarn install -g yahoo-finance-cli `
